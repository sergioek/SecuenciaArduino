Paso 1 - Realizar el montaje sencillo de un circuito en una protoboard

  • Placa Arduino: 
  • Protoboard: 
  • LED: 
  • Cable Dupont:
  • Resistencia: 

Colocamos un LED verde en la protoboard, como indica la figura.

Paso 2 - Programar los tiempos de encendido de nuestro led

Ejecutamos la aplicación Arduino, seleccionamos la placa y el puerto serie.

Una vez abierta la aplicación nos vamos a Herramientas/Tarjeta/Arduino UNO

Luego, seleccionaremos el puerto donde está conectado nuestra placa Arduino

Programacion en IDE Arduino

Copiamos el código del programa en el IDE de Arduino de tu computadora . 

void setup() {


pinMode(13,OUTPUT) ;


}


void loop() {
  digitalWrite(13, HIGH);


  delay(1000) ; 


  digitalWrite(13, LOW);


  delay(1000);


}

Una vez copiado presionamos  para cargar el programa en la placa Arduino .

Si seguiste los pasos del instructivo ,el led debe estar parpadeando . 

Pasemos a la clase N°2

Actividad

  • Cambia en el programa los tiempos de encendido y apagado .
  • modifica en el circuito el pin 13 por el pin12 y cambia la estructura del programa para que funcione no olvides presionar   para cargar el programa modificado en la placa.
