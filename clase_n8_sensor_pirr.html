CLASE N°8 Sensor Pirr

Sensor Pir

Para este circuito vamos a necesitar

Placa Arduino 

Cable dupont 

Sensor pir 

Protoboard 

Básicamente este sensor mide la luz infrarroja de los objetos en su campo de visión, por lo tanto, pueden detectar el movimiento en función de los cambios de esta luz en el entorno.

Y este es el esquema de conexión.

Codigo

Este es el codigo de nuestro sensor pir


const int pirPin = 2;  

void setup() {
  pinMode(pirPin, INPUT);
  Serial.begin(9600);
}

void loop() {
  int pirState = digitalRead(pirPin);

  if (pirState == HIGH) {
  
    Serial.println("Movimiento detectado");
  } else {
     
    Serial.println("Sin movimiento");
  }

  delay(100);  
}
