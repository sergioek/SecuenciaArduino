CLASE N°4 Sensor Ultrasonico

Sensor ultrasonico

Paso 1 - Realizamos el montaje de un sensor ultrasónico y 2 led a una protoboard

Paso 2 -Programar el sensor para que encienda los leds

Vamos a programar el sensor, para que detecte un obstáculo a 40 cm y prenda los led como de forma intermitente 

 float getDistance(int trig,int echo){

  pinMode(trig,OUTPUT);

  digitalWrite(trig,LOW);

  delayMicroseconds(2);

  digitalWrite(trig,HIGH);

  delayMicroseconds(10);

  digitalWrite(trig,LOW);

  pinMode(echo, INPUT);

  return pulseIn(echo,HIGH,30000)/58.0;

}

 void setup () {

 pinMode(2,OUTPUT) ;


 }

 void loop() {

 if((getDistance(8,9)) < ( 40)){

 digitalWrite(2,1);

 delay(100);

 digitalWrite(2,0);

 delay(100);

 digitalWrite(3,0);

 }else{

 digitalWrite(2,0);

}

}

Tarea

  1. El programa solo enciende una luz led, entonces haz que la secuencia active la luz verde para cuando esté abierto y luz roja cuando esté cerrado
  2. Cambiar la distancia donde el sensor activa los led
